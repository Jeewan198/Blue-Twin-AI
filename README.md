# Blue-Twin-AI
A Digital Twin Framework for Optimizing Osmotic Power Generation using Reinforcement Learning.

## Project Overview
This project aims to develop a software-based Digital Twin and an AI-driven control system that optimizes the water mixing process in an osmotic power plant (Reverse Electrodialysis) to increase net power density using Reinforcement Learning.

## Directory Structure
* `/data` - Data pipeline and storage for the ARA24 Global Salinity Database.
* `/simulation` - Mathematical modeling of the RED system (Gibbs free energy & Solution-Diffusion).
* `/agent` - Reinforcement Learning agent scripts (Stable Baselines3 / PyTorch).
* `/notebooks` - Exploratory data analysis and experimental logs.
